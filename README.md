# My C++ Project

## Build

```bash
cd build
cmake ..
cmake --build .
```

## Run

```bash
./build/main
```

## Development

Open with Neovim:
```bash
nvim .
```

### Key Bindings in Neovim
- `<Space>cb` - Build project
- `<Space>cr` - Run program
- `<Space>ch` - Switch between header/source
- `gd` - Go to definition
- `K` - Show documentation
